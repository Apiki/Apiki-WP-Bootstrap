Introdução ao Apiki WP Bootstrap

Apiki Flat UI é ridiculamente pequeno. Todo os modulos por apenas 17KB * minificado e compactado. Com grande facilidade de uso. Cada linha de CSS foi cuidadosamente considerada. Se você decidir usar apenas um subconjunto destes módulos, você vai economizar ainda mais bytes. :)
